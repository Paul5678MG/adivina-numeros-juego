# Adivina el Número - Paul Edition 🎮

Un juego interactivo desarrollado en **Python 3.12** que desafía al usuario a encontrar un número secreto dentro de un rango dinámico generado aleatoriamente.

## 🚀 Evolución del Proyecto
Este repositorio muestra el progreso de desarrollo desde una aplicación lógica simple hasta su evolución a una ejecutable:

- **v1 (Consola):** Lógica pura de Python, manejo de listas y estructuras de control.
- **v2 (GUI & Executable):** Implementación de interfaz gráfica con `Tkinter` y empaquetado final a .exe usando PyInstaller.

## 🛠️ Habilidades Técnicas Aplicadas
- **Bibliotecas:** `tkinter`, `random`, `messagebox`.
- **Lógica de Programación:** Generación de listas dinámicas sin duplicados, ordenamiento de datos y manejo de estados de botones.
- **Robustez:** Implementación de validación de datos para evitar errores de tipo `ValueError` si el usuario ingresa texto en lugar de números.
- **Recursos Visuales:** Diseño y gestión de activos (.ico) para personalización de software.

## 💾 Descarga el Ejecutable (.exe)
Si no tienes Python instalado, puedes descargar la versión lista para jugar (con icono personalizado) desde la sección de **[Releases](https://github.com/Paul5678MG/adivina-numeros-juego/releases)** de este repositorio.

## 📦 Instalación y Uso para desarrolladores
1. Clona el repositorio:
   `git clone https://github.com/Paul5678MG/adivina-numeros-juego.git`
2. Ejecuta la versión de tu preferencia:
   `python v1_console/adivinar_num.py`
   `python v2_gui/adivinar_num_v2.py`
